--- tag: components section: Spacing code-language: markup show-example: true show-code: true description: > We are using Bootstrap v4’s new spacing classes for when you need extra padding or margin for an element. You can read about it more in their documents. For Bricks, we have also added corresponding negative margins to Bootstrap's classes. So if you want an element to have margin-right: 1rem !important;, you'd use .mr-3 and if you want an element to have margin-right: -1rem !important;, you'd use .mr--3. ---
